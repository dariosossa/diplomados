Manual de Construcción: Página de Login

Construiremos una vista de inicio de sesión moderna con pantalla dividida, analizando línea por línea cómo se hace en CSS Clásico y en Tailwind.

0. Estructura Básica del Documento

Preparamos el archivo HTML base.

Método 1: HTML y CSS Clásico

<!-- Archivo: login.html -->
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Iniciar Sesión - Valerius Abogados</title>
  
  <!-- Vincula el archivo CSS externo que contendrá nuestras reglas -->
  <link rel="stylesheet" href="login-style.css">
</head>
<body>
  <!-- El contenido irá aquí -->
</body>
</html>

/* Archivo: login-style.css */
/* El asterisco (*) selecciona TODOS los elementos de la página. 
   Es el famoso "CSS Reset" para evitar comportamientos inesperados del navegador. */
* { 
  margin: 0; /* Elimina márgenes externos por defecto */
  padding: 0; /* Elimina espaciados internos por defecto */
  box-sizing: border-box; /* Asegura que el padding/borde se incluya en el ancho total del elemento (no lo hace más grande) */
}

/* Establece la tipografía base para toda la página */
body { 
  font-family: sans-serif; 
}

Método 2: Tailwind CSS

<!-- Archivo: login.html -->
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Iniciar Sesión - Valerius Abogados</title>

  <script src="https://cdn.tailwindcss.com"></script>
</head>
<!-- 
  font-sans: Aplica la tipografía sans-serif por defecto de Tailwind.
  antialiased: Suaviza los bordes de las letras para que se vean mejor en pantallas Mac/iOS.
  bg-white: Fondo blanco para toda la página. 
-->
<body class="font-sans antialiased bg-white">
  <!-- El contenido irá aquí -->
</body>
</html>

1. Barra de Navegación Flotante

Un encabezado simple que se posiciona "flotando" encima de la imagen de fondo.

Método 1: HTML y CSS Clásico

<header class="nav-login">
  <div class="logo">
    <span class="marca">Valerius</span>
  </div>
  <a href="index.html" class="btn-volver">← Volver al Inicio</a>
</header>

<style>
.nav-login { 
  position: absolute; /* Saca al elemento del flujo normal, permitiendo que otros elementos se pongan debajo */
  top: 0; /* Lo pega al borde superior de la pantalla */
  left: 0; /* Lo pega al borde izquierdo */
  width: 100%; /* Ocupa todo el ancho de la pantalla */
  display: flex; /* Activa el modelo de caja flexible para alinear los hijos (logo y botón) */
  justify-content: space-between; /* Empuja el logo a la izq. y el botón a la der., dejando el espacio al medio */
  align-items: center; /* Centra los elementos verticalmente */
  padding: 20px 5%; /* Da espacio interior: 20px arriba/abajo, 5% del ancho a los lados */
  z-index: 10; /* Asegura que este elemento siempre esté "por encima" de la imagen de fondo */
}

.logo .marca { 
  font-size: 1.6rem; /* Tamaño de letra grande */
  color: #c5a059; /* Color dorado institucional */
  font-weight: 700; /* Letra en negrita (bold) */
}

.btn-volver { 
  color: #ffffff; /* Texto blanco */
  text-decoration: none; /* Quita la línea de subrayado típica de los enlaces */
  font-weight: 600; /* Letra semi-negrita */
  background: rgba(0,0,0,0.5); /* Fondo negro con 50% de transparencia (el 0.5 final) */
  padding: 8px 16px; /* Espaciado interno para darle forma de botón */
  border-radius: 20px; /* Bordes muy redondeados (tipo píldora) */
  transition: background 0.3s, color 0.3s; /* La animación del cambio de color tardará 0.3 segundos */
}

/* El pseudo-elemento :hover se activa cuando el ratón pasa por encima */
.btn-volver:hover { 
  background: #c5a059; /* Cambia el fondo a dorado */
  color: black; /* Cambia el texto a negro para mantener buen contraste */
}
</style>

Método 2: Tailwind CSS

<!-- 
  absolute: Lo posiciona independientemente (flotando).
  top-0, left-0, w-full: Lo ancla arriba, a la izquierda y ocupando el 100% del ancho.
  flex: Activa flexbox.
  justify-between: Separa logo a la izq. y botón a la der.
  items-center: Centrado vertical.
  px-[5%]: Relleno (padding) en el eje X (izq/der) de un 5%.
  py-5: Relleno (padding) en el eje Y (arriba/abajo) de un tamaño predefinido (1.25rem).
  z-10: Lo pone por encima de otros elementos (índice Z).
-->
<header class="absolute top-0 left-0 w-full flex justify-between items-center px-[5%] py-5 z-10">
  
  <!-- 
    text-[#c5a059]: Color dorado hexadecimal personalizado.
    text-[1.6rem]: Tamaño de fuente personalizado.
    font-bold: Negrita.
    drop-shadow-md: Pequeña sombra al texto para que resalte sobre imágenes complejas.
  -->
  <div class="text-[#c5a059] text-[1.6rem] font-bold drop-shadow-md">
    Valerius
  </div>

  <!-- 
    text-white: Texto blanco.
    font-semibold: Semi-negrita.
    bg-black/50: Fondo negro con 50% de opacidad.
    px-4, py-2: Relleno horizontal y vertical.
    rounded-full: Borde totalmente redondeado (píldora).
    backdrop-blur-sm: Desenfoque del fondo detrás del botón (efecto cristal/glassmorphism).
    hover:bg-[#c5a059], hover:text-black: Cambios al pasar el ratón.
    transition-all: Anima todos los cambios de propiedades suavemente.
  -->
  <a href="index.html" class="text-white font-semibold bg-black/50 px-4 py-2 rounded-full backdrop-blur-sm hover:bg-[#c5a059] hover:text-black transition-all">
    &larr; Volver al Inicio
  </a>

</header>

2. Diseño de Pantalla Dividida (Split Screen)

Un diseño donde la imagen ocupa la mitad izquierda y el formulario la mitad derecha en pantallas de PC. En celulares, la imagen se oculta.

Método 1: HTML y CSS Clásico

<main class="contenedor-split">
  
  <!-- Contenedor de la imagen (Izquierda) -->
  <div class="mitad-imagen"></div>

  <!-- Contenedor del formulario (Derecha) -->
  <div class="mitad-formulario">
    <div class="caja-login">
      <h2>Bienvenido de nuevo</h2>
      <p class="subtitulo">Ingrese sus credenciales para acceder a su portal.</p>

      <form>
        <label>Correo Electrónico</label>
        <input type="email" placeholder="ejemplo@correo.com" required>

        <label>Contraseña</label>
        <input type="password" placeholder="••••••••" required>

        <div class="opciones-extra">
          <label class="recordarme"><input type="checkbox"> Recordarme</label>
          <a href="#">¿Olvidó su contraseña?</a>
        </div>

        <button type="submit" class="btn-login">Ingresar</button>
      </form>
      
      <p class="registro">¿No tiene una cuenta? <a href="#">Solicite acceso</a></p>
    </div>
  </div>

</main>

<style>
/* Contenedor principal de la pantalla dividida */
.contenedor-split { 
  display: flex; /* Permite que los hijos (.mitad-imagen y .mitad-formulario) se coloquen uno al lado del otro */
  min-height: 100vh; /* La altura mínima será el 100% de la ventana del navegador (Viewport Height) */
}

/* Área de la imagen */
.mitad-imagen { 
  flex: 1; /* Le dice que ocupe una fracción del espacio disponible (la mitad, si el otro elemento también tiene flex: 1) */
  /* Fondo: primero un gradiente oscuro semitransparente, luego la imagen. El orden importa. */
  background: linear-gradient(rgba(0,0,0,0.5), rgba(0,0,0,0.5)), url('img.jpg'); 
  background-size: cover; /* La imagen se estira o recorta para cubrir todo el espacio sin deformarse */
  background-position: center; /* Centra la parte más importante de la imagen */
  display: none; /* POR DEFECTO OCULTO (Estrategia "Mobile First", primero pensamos en celulares) */
}

/* Área del formulario */
.mitad-formulario { 
  flex: 1; /* Ocupa la otra mitad del espacio (cuando la imagen sea visible) */
  display: flex; /* Usamos flex aquí para poder centrar la caja interior */
  align-items: center; /* Centra la caja verticalmente */
  justify-content: center; /* Centra la caja horizontalmente */
  background: #ffffff; /* Fondo blanco para la zona del formulario */
  padding: 2rem; /* Espacio interior para que nada pegue a los bordes */
}

/* La caja central que contiene los inputs */
.caja-login { 
  width: 100%; /* Ocupa todo el ancho que le dejen... */
  max-width: 400px; /* ...pero nunca será más ancha de 400px (evita inputs larguísimos en PC) */
}

/* Estilos de tipografía básica */
.caja-login h2 { font-family: "Playfair Display", serif; font-size: 2.5rem; color: #0f172a; margin-bottom: 0.5rem; }
.caja-login .subtitulo { color: #64748b; margin-bottom: 2rem; }
.caja-login label { display: block; /* Ocupa su propia línea encima del input */ font-weight: 600; color: #1e293b; margin-bottom: 0.5rem; font-size: 0.9rem; }

/* Estilos de las cajas de texto (inputs) */
.caja-login input[type="email"], .caja-login input[type="password"] { 
  width: 100%; /* Ocupan todo el ancho del contenedor */
  padding: 15px; /* Espacio interno para que sean cajas altas y cómodas de clickear */
  margin-bottom: 1.5rem; /* Separación con el siguiente elemento */
  border: 1px solid #cbd5e1; /* Borde gris claro sutil */
  border-radius: 8px; /* Esquinas ligeramente redondeadas */
  outline: none; /* Quita el borde azul grueso que ponen los navegadores al hacer clic por defecto */
  transition: 0.3s; /* Animación para cuando el usuario haga clic */
}

/* Estado Focus: Cuando el usuario hace clic dentro de la caja para escribir */
.caja-login input:focus { 
  border-color: #c5a059; /* El borde cambia a color dorado */
  box-shadow: 0 0 0 3px rgba(197, 160, 89, 0.2); /* Crea un "aura" dorada suave simulando un borde más grueso */
}

/* Opciones extra: Recordarme y Contraseña olvidada */
.opciones-extra { 
  display: flex; 
  justify-content: space-between; /* El checkbox a un lado, el enlace al otro */
  align-items: center; /* Alineados verticalmente en la misma línea */
  margin-bottom: 2rem; 
  font-size: 0.9rem; 
}
.opciones-extra a { color: #c5a059; text-decoration: none; font-weight: 600; }

/* Casilla de Recordarme */
.recordarme { 
  display: flex; 
  align-items: center; /* Alinea el cuadrito con el texto */
  gap: 8px; /* Separación entre el cuadrito y la palabra */
  font-weight: normal; 
  margin-bottom: 0; 
}

/* Botón principal de Iniciar Sesión */
.btn-login { 
  width: 100%; 
  background: #0f172a; /* Color azul muy oscuro */
  color: white; 
  padding: 15px; 
  border: none; /* Quita bordes por defecto de botones html */
  border-radius: 8px; 
  font-weight: bold; 
  font-size: 1.1rem; 
  cursor: pointer; /* Cambia el cursor a una 'manito' */
  transition: 0.3s; 
}
.btn-login:hover { background: #c5a059; color: black; } /* Cambio de color al pasar el mouse */

/* Textos finales */
.registro { text-align: center; margin-top: 2rem; color: #64748b; font-size: 0.95rem; }
.registro a { color: #0f172a; font-weight: bold; text-decoration: none; }

/* MEDIA QUERY: "Media" son condiciones de pantalla. 
   "min-width: 1024px" significa: si la pantalla es de 1024px de ancho o MÁS grande (Tablets/PCs),
   entonces aplica las reglas que están adentro. */
@media (min-width: 1024px) {
  .mitad-imagen { 
    display: block; /* Pasa de "none" (oculto) a "block" (visible). Ahora ocupará la mitad de la pantalla. */
  }
}
</style>

Método 2: Tailwind CSS

Tailwind simplifica muchísimo la adaptabilidad móvil (Responsive Design). Los prefijos como lg: o sm: actúan como "Media Queries" en línea.

<!-- 
  min-h-screen: Altura mínima 100vh.
  flex: Activa flexbox (fila de contenedores).
-->
<main class="min-h-screen flex">
  
  <!-- 
    MITAD IZQUIERDA (IMAGEN)
    - hidden: (Móviles) Oculta la imagen por defecto.
    - lg:block: (Pantallas Grandes) Muestra la imagen (display: block).
    - lg:w-1/2: (Pantallas Grandes) Fuerza a que ocupe el 50% (1/2) del ancho disponible.
    - bg-[linear-gradient(...)]: Inserta color oscuro sobre la foto.
    - bg-cover, bg-center: Ajusta la foto sin deformar y centrada.
  -->
  <div class="hidden lg:block lg:w-1/2 bg-[linear-gradient(rgba(0,0,0,0.6),rgba(0,0,0,0.6)),url('https://estaticos-cdn.prensaiberica.es/clip/3c5e38ad-fa96-49ae-b706-872d87b181ab_alta-libre-aspect-ratio_default_0.jpg')] bg-cover bg-center"></div>

  <!-- 
    MITAD DERECHA (FORMULARIO)
    - w-full: En celulares ocupa el 100% de la pantalla (ya que la imagen está oculta).
    - lg:w-1/2: En PC ocupa solo la mitad derecha.
    - flex items-center justify-center: Centra el contenido internamente, tanto vertical como horizontalmente.
    - bg-white: Fondo blanco.
    - p-8 sm:p-12: Padding base de 2rem, pero si es pantalla pequeña (sm) o más, crece a 3rem.
  -->
  <div class="w-full lg:w-1/2 flex items-center justify-center bg-white p-8 sm:p-12">
    
    <!-- 
      w-full max-w-md: La caja puede ocupar el 100% del padre, 
      pero nunca superará el tamaño 'Medium' (28rem / 448px), evitando inputs gigantes en PC. 
    -->
    <div class="w-full max-w-md">
      
      <h2 class="font-serif text-4xl font-bold text-slate-900 mb-2">Bienvenido de nuevo</h2>
      <p class="text-slate-500 mb-8">Ingrese sus credenciales para acceder a su portal.</p>

      <!-- flex-col: Los elementos del formulario (labels e inputs) se apilarán en columna, uno debajo del otro -->
      <form class="flex flex-col">
        
        <label class="font-semibold text-slate-700 text-sm mb-2">Correo Electrónico</label>
        <!-- 
          ESTILO DE INPUTS:
          - w-full, p-4: Ancho completo, padding interno grande.
          - mb-6: Margen inferior para separarlo del siguiente label.
          - border border-slate-300: Borde de 1px gris clarito.
          - rounded-lg: Esquinas redondeadas (Large).
          - outline-none: Quita el borde azul molesto del navegador al hacer clic.
          (EFECTO HOVER/FOCUS)
          - focus:border-[#c5a059]: Al hacer clic (focus), el borde cambia a dorado.
          - focus:ring-4 focus:ring-[#c5a059]/20: Al hacer clic, crea un "anillo" de 4px alrededor, dorado pero con 20% de opacidad (aura sutil).
          - transition-all: Hace que el paso a Focus sea animado, no brusco.
        -->
        <input type="email" placeholder="ejemplo@correo.com" class="w-full p-4 mb-6 border border-slate-300 rounded-lg outline-none focus:border-[#c5a059] focus:ring-4 focus:ring-[#c5a059]/20 transition-all" required>

        <label class="font-semibold text-slate-700 text-sm mb-2">Contraseña</label>
        <input type="password" placeholder="••••••••" class="w-full p-4 mb-6 border border-slate-300 rounded-lg outline-none focus:border-[#c5a059] focus:ring-4 focus:ring-[#c5a059]/20 transition-all" required>

        <!-- 
          flex justify-between items-center: Manda el checkbox a la izq y el link a la derecha, centrados verticalmente en la misma línea. 
        -->
        <div class="flex justify-between items-center mb-8 text-sm">
          
          <!-- flex items-center gap-2: Alinea el cuadrito y el texto horizontalmente, separados por 'gap' -->
          <label class="flex items-center gap-2 text-slate-600 cursor-pointer">
            <!-- accent-[#c5a059]: (Propiedad moderna) Cambia el color del 'check' azul por defecto a dorado -->
            <input type="checkbox" class="w-4 h-4 accent-[#c5a059]"> Recordarme
          </label>
          
          <a href="#" class="text-[#c5a059] font-bold hover:underline">¿Olvidó su contraseña?</a>
        </div>

        <!-- 
          ESTILO DEL BOTÓN:
          bg-[#0f172a] text-white: Fondo azul oscuro, texto blanco.
          hover:bg-[#c5a059] hover:text-black: Al pasar el ratón, fondo dorado, texto negro. 
        -->
        <button type="submit" class="w-full bg-[#0f172a] text-white font-bold p-4 rounded-lg hover:bg-[#c5a059] hover:text-black transition-colors">
          Ingresar
        </button>
        
      </form>

      <!-- text-center mt-8: Texto centrado, con margen arriba para separarlo del botón -->
      <p class="text-center mt-8 text-slate-500 text-sm">
        ¿No tiene una cuenta? <a href="#" class="text-[#0f172a] font-bold hover:underline">Solicite acceso</a>
      </p>

    </div>
  </div>

</main>
