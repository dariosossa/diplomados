Manual de Construcción: Valerius Abogados

Aprende a construir una Landing Page profesional comparando CSS Clásico y Tailwind CSS paso a paso.

0. Estructura Básica del Documento

Antes de comenzar a escribir el código de nuestro diseño, necesitamos preparar el archivo HTML base para que el navegador lo entienda y reconozca nuestros estilos.

Método 1: HTML y CSS Clásico

En el método tradicional, creamos un archivo HTML y lo vinculamos a un archivo CSS externo mediante la etiqueta <link>.

<!-- Archivo: index.html -->
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Valerius Abogados</title>
  
  <!-- Enlace a nuestro archivo CSS externo -->
  <link rel="stylesheet" href="style.css">
</head>
<body>
  
  <!-- Aquí irá el contenido de nuestra página -->

</body>
</html>


/* Archivo: style.css (Reset básico de márgenes) */
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}
body {
  font-family: sans-serif;
}

Método 2: Tailwind CSS

Para usar Tailwind rápidamente sin instalar programas en tu computadora, utilizamos su versión CDN (Content Delivery Network). Solo necesitas pegar un <script> en el encabezado de tu HTML. ¡No necesitas un archivo CSS extra!

<!-- Archivo: index.html -->
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Valerius Abogados - Tailwind</title>
  
  <!-- SCRIPT OBLIGATORIO: Con esto Tailwind funcionará en tu navegador -->
  <script src="https://cdn.tailwindcss.com"></script>
</head>
<body class="font-sans antialiased">
  
  <!-- Aquí irá el contenido con las clases utilitarias de Tailwind -->

</body>
</html>
