Código Explicado al Detalle: index-admin.html

Este es el archivo principal del panel que contiene la barra lateral (Sidebar) y el iframe. Cada propiedad CSS y cada clase de Tailwind está comentada para el aula.

Opción 1: HTML y CSS Clásico

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Panel de Control - Valerius Abogados</title>
  
  <style>
    /* =========================================
       1. RESET Y ESTILOS BASE
       ========================================= */
    * { 
      margin: 0;               /* Quita márgenes externos de todos los elementos */
      padding: 0;              /* Quita rellenos internos de todos los elementos */
      box-sizing: border-box;  /* Evita que los bordes modifiquen el ancho/alto de las cajas */
    }
    body { 
      font-family: sans-serif; /* Tipografía general limpia */
      background-color: #f8fafc; /* Fondo general gris clarito */
    }

    /* =========================================
       2. ESTRUCTURA PRINCIPAL (FLEXBOX)
       ========================================= */
    .admin-layout { 
      display: flex;           /* Activa Flexbox para poner la barra a la izq y el iframe a la der */
      height: 100vh;           /* Altura estricta del 100% de la ventana visible del navegador */
      overflow: hidden;        /* Oculta cualquier cosa que se salga de la pantalla (evita scroll doble) */
    }

    /* =========================================
       3. BARRA LATERAL (SIDEBAR)
       ========================================= */
    .sidebar { 
      width: 260px;            /* Ancho estricto y fijo para la barra lateral */
      background: #0f172a;     /* Fondo azul marino muy oscuro */
      color: white;            /* Todo el texto dentro será blanco por defecto */
      display: flex;           /* Flexbox interno en la barra lateral */
      flex-direction: column;  /* Apila los elementos en columna (arriba hacia abajo) */
      padding: 20px;           /* Relleno interno de 20px en los 4 lados */
    }
    
    /* Logotipo de la Barra Lateral */
    .sidebar-logo { 
      font-size: 1.5rem;       /* Tamaño de letra grande (24px aprox) */
      color: #c5a059;          /* Letras color dorado institucional */
      font-weight: 700;        /* Letra muy en negrita */
      margin-bottom: 2rem;     /* Empuja el menú de abajo separándolo 32px */
      border-bottom: 1px solid #1e293b; /* Línea separadora inferior de color gris oscuro */
      padding-bottom: 1rem;    /* Espacio entre el texto del logo y la línea separadora */
    }
    
    /* Contenedor del Menú de Navegación */
    .sidebar-menu { 
      flex-grow: 1;            /* ¡CLAVE! Hace que este contenedor ocupe TODO el espacio vertical libre, empujando el botón de salir hacia el fondo */
      display: flex;           /* Activa Flexbox para los enlaces */
      flex-direction: column;  /* Apila los enlaces uno debajo del otro */
      gap: 10px;               /* Separación exacta de 10px entre cada enlace */
    }
    
    /* Enlaces del Menú */
    .sidebar-menu a { 
      color: #cbd5e1;          /* Texto gris clarito (no blanco puro) para no cansar la vista */
      text-decoration: none;   /* Quita la línea inferior típica de los links */
      padding: 12px 15px;      /* Alto y ancho interno para que parezcan botones */
      border-radius: 8px;      /* Esquinas redondeadas */
      transition: 0.3s;        /* Animación suave de colores al hacer hover */
    }
    .sidebar-menu a:hover { 
      background: #1e293b;     /* Fondo gris oscuro al pasar el ratón */
      color: white;            /* Texto blanco brillante al pasar el ratón */
    }
    
    /* Enlace Activo (La página donde estamos parados) */
    .sidebar-menu a.activo { 
      background: #c5a059;     /* Fondo dorado */
      color: black;            /* Texto negro */
      font-weight: bold;       /* Negrita para resaltar */
    }
    
    /* Botón Cerrar Sesión (Ubicado al final de la barra) */
    .btn-salir { 
      color: #f87171;          /* Texto color rojo pastel */
      text-decoration: none;   /* Quita subrayado */
      font-weight: 600;        /* Semi-negrita */
      padding: 12px 15px;      /* Mismas medidas que los enlaces del menú */
    }

    /* =========================================
       4. ÁREA PRINCIPAL E IFRAME
       ========================================= */
    .area-principal { 
      flex: 1;                 /* Ocupa TODO el ancho sobrante de la pantalla que la barra lateral no usa */
      padding: 1.5rem;         /* Espacio interno de 24px alrededor del iframe */
    }
    
    .iframe-contenido { 
      width: 100%;             /* Ocupa el 100% del ancho del .area-principal */
      height: 100%;            /* Ocupa el 100% de la altura del .area-principal */
      border: none;            /* Quita el borde nativo, grueso y feo de los iframes */
      border-radius: 12px;     /* Redondea las esquinas del iframe */
      background: white;       /* Fondo blanco detrás de los contenidos del iframe */
      box-shadow: 0 4px 6px -1px rgba(0,0,0,0.05); /* Sombra muy sutil para que parezca una tarjeta flotando sobre el fondo gris */
    }
  </style>
</head>
<body>

  <!-- 2. ESTRUCTURA PRINCIPAL -->
  <div class="admin-layout">
    
    <!-- 3. BARRA LATERAL -->
    <aside class="sidebar">
      <div class="sidebar-logo">Valerius Admin</div>
      
      <!-- MENÚ: Contiene los enlaces con 'target' hacia el iframe -->
      <nav class="sidebar-menu">
        <!-- El atributo target="visor_crud" abre esta URL dentro del iframe -->
        <a href="casos.html" target="visor_crud" class="activo">Gestión de Casos</a>
        <a href="clientes.html" target="visor_crud">Clientes</a>
        <a href="facturas.html" target="visor_crud">Facturación</a>
      </nav>
      
      <!-- Botón inferior para salir del sistema -->
      <a href="login.html" class="btn-salir">Cerrar Sesión</a>
    </aside>

    <!-- 4. ÁREA PRINCIPAL (IFRAME) -->
    <main class="area-principal">
      <!-- 
        name="visor_crud": Identificador que buscan los enlaces 'target'.
        src="casos.html": Página que se carga por defecto al entrar al panel. 
      -->
      <iframe name="visor_crud" src="casos.html" class="iframe-contenido" title="Visor de Datos"></iframe>
    </main>

  </div>

</body>
</html>

Opción 2: Tailwind CSS

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Panel de Control - Tailwind</title>
  
  <!-- CARGA DE LIBRERÍA CDN -->
  <script src="https://cdn.tailwindcss.com"></script>
</head>

<!-- 
  ================= 1. ESTILOS BASE DEL BODY =================
  font-sans: Tipografía sans-serif.
  antialiased: Suavizado de bordes en Mac/iOS.
  bg-slate-50: Color de fondo gris extraclaro (#f8fafc) para todo el panel.
  text-slate-800: Color de texto general oscuro.
-->
<body class="font-sans antialiased bg-slate-50 text-slate-800">

  <!-- 
    ================= 2. ESTRUCTURA PRINCIPAL =================
    flex: Pone la barra lateral y el iframe uno al lado del otro en horizontal.
    h-screen: height: 100vh. Altura exacta de la ventana.
    overflow-hidden: Impide que haya barra de desplazamiento general en el navegador.
  -->
  <div class="flex h-screen overflow-hidden">
    
    <!-- 
      ================= 3. BARRA LATERAL (SIDEBAR) =================
      w-64: width de 16rem (256px). Ancho fijo de la barra.
      bg-[#0f172a]: Fondo azul marino oscuro.
      text-white: Letra blanca por defecto.
      flex flex-col: Alineación en columna (apila logo, menú y botón salir hacia abajo).
      p-6: padding de 1.5rem (24px) alrededor del borde interno.
    -->
    <aside class="w-64 bg-[#0f172a] text-white flex flex-col p-6">
      
      <!-- 
        LOGOTIPO
        text-[#c5a059]: Letra dorada.
        text-2xl font-bold: Tamaño grande y negrita.
        mb-8: margin-bottom de 32px (separa del menú).
        pb-4: padding-bottom de 16px (separa el texto de la línea de abajo).
        border-b border-slate-800: Aplica un borde inferior de 1px color gris oscuro.
      -->
      <div class="text-[#c5a059] text-2xl font-bold mb-8 pb-4 border-b border-slate-800">
        Valerius Admin
      </div>
      
      <!-- 
        MENÚ DE ENLACES
        flex-grow: Esta clase es VITAL. Hace que el 
 crezca para ocupar todo el 
                   espacio sobrante en la barra lateral. Eso obliga al botón de "Cerrar Sesión" 
                   a quedarse aplastado en la parte inferior.
        flex flex-col gap-2: Apila los  uno sobre otro, con separación de 8px (0.5rem).
      -->
      <nav class="flex-grow flex flex-col gap-2">
        
        <!-- 
          ENLACE ACTIVO (Donde estamos parados)
          bg-[#c5a059]: Fondo dorado.
          text-black font-bold: Letra negra y gorda.
          px-4 py-3: padding interno.
          rounded-lg: Borde redondo 8px.
        -->
        <a href="casos.html" target="visor_crud" class="bg-[#c5a059] text-black font-bold px-4 py-3 rounded-lg">
          Gestión de Casos
        </a>
        
        <!-- 
          ENLACE INACTIVO
          text-slate-300: Texto gris claro normal.
          px-4 py-3 rounded-lg: Mismo tamaño y forma.
          hover:bg-slate-800 hover:text-white: Al pasar ratón se pone gris oscuro con texto blanco vivo.
          transition-colors: Animación.
        -->
        <a href="clientes.html" target="visor_crud" class="text-slate-300 px-4 py-3 rounded-lg hover:bg-slate-800 hover:text-white transition-colors">
          Clientes
        </a>
        <a href="facturas.html" target="visor_crud" class="text-slate-300 px-4 py-3 rounded-lg hover:bg-slate-800 hover:text-white transition-colors">
          Facturación
        </a>

      </nav>

      <!-- 
        BOTÓN SALIR
        text-red-400 font-bold: Letra roja pastel y gruesa.
        px-4 py-3: Padding interno.
        hover:text-red-300: Ratón encima pone el rojo más claro.
        mt-auto: margin-top: auto. Asegura que siempre esté pegado al borde inferior.
      -->
      <a href="login.html" class="text-red-400 font-bold px-4 py-3 hover:text-red-300 mt-auto">
        Cerrar Sesión
      </a>
    </aside>

    <!-- 
      ================= 4. ÁREA DEL IFRAME =================
      flex-1: Toma el resto del ancho de la pantalla (100% menos los 256px de la barra lateral).
      p-6: Padding de 24px en todos los lados.
    -->
    <main class="flex-1 p-6">
      
      <!-- 
        IFRAME DE CONTENIDO
        name="visor_crud": Recibe la orden de los "target" del menú.
        src="casos.html": Archivo que carga por defecto.
        w-full h-full: Ocupa el 100% de la etiqueta 
.
        border-none: Quita borde gris nativo.
        rounded-xl: Borde extra redondeado (12px).
        shadow-sm: Sombra sutil.
        bg-white: Fondo blanco.
      -->
      <iframe name="visor_crud" src="casos.html" class="w-full h-full border-none rounded-xl shadow-sm bg-white"></iframe>
      
    </main>

  </div>

</body>
</html>
