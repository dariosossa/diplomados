¿Qué esperan del diplomado?

¿Que es el desarrollo Web?

El desarrollo web es el proceso de crear y mantener sitios o aplicaciones que funcionan en internet.

¿Que es el desarrollo Móvil?

El desarrollo móvil es el proceso de crear aplicaciones diseñadas para funcionar en dispositivos móviles.

Introducción al Desarrollo Web y Móvil

¿Qué es un sistema operativo?

Un sistema operativo es el “software” (sof-güer) principal que controla y administra los recursos de un dispositivo.

Se estudiarán los sistemas operativos Android, iOS, Windows y Linux.

¿Qué es Android?

Es un sistema operativo para dispositivos móviles.
Fue desarrollado por Google y está basado en Linux.

¿Qué es iOS (“ai-ou-es”)?

Es el sistema operativo desarrollado por Apple para sus dispositivos móviles.

¿Qué es Windows (“uín-dous”)?

Windows es un sistema operativo creado por Microsoft, permite utilizar programas y conectarse a internet.

¿Qué es Linux (“lí-nux”)?

Linux es un sistema operativo libre y de código abierto, creado inicialmente por Linus Torvalds en 1991.

Se utilizarán diferentes dispositivos, como computadoras, laptops, tabletas y teléfonos móviles.

¿Qué es una computadora?

Es un dispositivo electrónico capaz de recibir, procesar y almacenar información.

¿Qué es una laptop?

Es una computadora diseñada para ser transportada fácilmente y utilizada en diferentes lugares.

¿Qué es una tableta?

Es un dispositivo con pantalla táctil que permite interactuar directamente con la pantalla.

¿Qué son los teléfonos móviles?

Son dispositivos electrónicos portátiles que permiten la comunicación inalámbrica entre personas.

Evolución del desarrollo web

¿Cómo era la Web 1.0?

La Web 1.0 se caracterizaba por estar formada por páginas web estáticas, donde la información era publicada por los administradores del sitio y los usuarios solo podían leer el contenido sin interactuar con él.

Las páginas estaban compuestas principalmente por texto, imágenes y enlaces, y su función principal era mostrar información, similar a una biblioteca digital.

¿Cómo era la Web 2.0?

La Web 2.0 se caracterizó por permitir la interacción y participación activa de los usuarios en internet.

A diferencia de la Web 1.0, donde los usuarios solo podían leer información, en la Web 2.0 las personas pueden crear, compartir y comentar contenido en diferentes plataformas digitales.

En esta etapa surgieron servicios como redes sociales, blogs, wikis, plataformas de video y aplicaciones web interactivas, donde los usuarios se convierten también en creadores de contenido.

Ejemplos claros de Web 2.0 son plataformas como Facebook, YouTube y Twitter, donde la información es generada y compartida por millones de usuarios.

¿Cómo es la actual Web 3.0?

La Web 3.0 se caracteriza por ser una web más inteligente, conectada y personalizada, donde las aplicaciones pueden analizar grandes cantidades de información para ofrecer resultados más precisos a los usuarios.

En esta etapa de internet se utilizan tecnologías como inteligencia artificial, procesamiento de datos, computación en la nube y sistemas semánticos, que permiten que las plataformas comprendan mejor la información y el comportamiento de las personas.

Evolución del desarrollo móvil

¿Cómo fue la primera etapa?

La primera etapa del desarrollo móvil se caracterizó por el uso de teléfonos móviles básicos que tenían funciones limitadas, principalmente llamadas telefónicas y envío de mensajes de texto (SMS).

En ese periodo, los dispositivos tenían pantallas pequeñas, poca capacidad de almacenamiento y recursos de procesamiento muy reducidos, por lo que las aplicaciones eran simples y con pocas funciones.

¿Cómo fue la segunda etapa?

La segunda etapa del desarrollo móvil se caracterizó por la aparición de los primeros teléfonos inteligentes o smartphones, que comenzaron a ofrecer más funciones además de llamadas y mensajes.

En esta etapa los dispositivos incorporaron pantallas más grandes, conexión a internet, cámaras y sistemas operativos más avanzados, lo que permitió el desarrollo de aplicaciones más complejas.

También surgieron las primeras tiendas de aplicaciones, donde los usuarios podían descargar diferentes programas para ampliar las funciones de sus dispositivos.

¿Cómo es la etapa actual?

La etapa actual del desarrollo móvil se caracteriza por el uso de teléfonos inteligentes avanzados con gran capacidad de procesamiento, conexión permanente a internet y aplicaciones cada vez más complejas.

En esta etapa los dispositivos móviles permiten realizar múltiples actividades como comunicación, comercio electrónico, educación en línea, redes sociales, entretenimiento y trabajo remoto.

Además, las aplicaciones móviles actuales integran tecnologías como inteligencia artificial, computación en la nube, geolocalización y servicios digitales en tiempo real, lo que permite ofrecer experiencias más rápidas y personalizadas a los usuarios.

Arquitectura de Aplicaciones Web y Móviles

¿Qué es la arquitectura de una aplicación?

Es la forma en que se organiza un sistema.

¿Qué es el frontend (“front-end”)?

Es la parte de una aplicación o página web que el usuario puede ver y con la que puede interactuar.

¿Qué es el backend (“bak-end”)?

Es la parte de una aplicación que funciona en el servidor y que el usuario no ve.

Tipos de aplicaciones

¿Qué son las aplicaciones nativas?

Son programas desarrollados específicamente para un sistema operativo.

¿Qué son las aplicaciones híbridas?

Son aplicaciones desarrolladas con tecnologías web, pero que pueden instalarse en dispositivos móviles.

¿Qué son las aplicaciones web?

Son programas que funcionan a través de un navegador de internet, sin necesidad de instalarlos.

¿Qué es el cliente–servidor?

Es un modelo de comunicación donde se comunica un dispositivo cliente con otro llamado servidor.

¿Qué es MVC?

Es un patrón de arquitectura de software que organiza una aplicación en tres partes.

¿Qué es MVVM?

Es un patrón de arquitectura de software que organiza una aplicación en tres partes.

Diferencia entre Controlador (MVC) y ViewModel (MVVM)

El Controlador responde a eventos del usuario y controla el flujo de la aplicación, mientras que el ViewModel prepara los datos y mantiene sincronizada la vista con el modelo.

¿Qué es SPA (“es-pi-ei”)?

Es una aplicación de una sola página.

¿Qué es HTML (“ei-ch-ti-em-el”)?

Es el lenguaje estándar que se utiliza para crear y estructurar páginas web.

HTML permite organizar el contenido de una página mediante etiquetas, indicando qué es cada elemento, por ejemplo:

  • Títulos
  • Párrafos
  • Enlaces
  • Imágenes
  • Listas
  • Tarjetas
  • Tablas

¿Qué es la estructura básica de HTML?

La estructura básica de HTML es la organización inicial que debe tener toda página web para que el navegador pueda interpretarla correctamente.

Esta estructura define el tipo de documento, la configuración de la página y el contenido que se mostrará al usuario.

Ejemplo de estructura básica:

<!DOCTYPE html>
<html>

    <head>
        <style>
            body {
                background-color: white; /* Fondo blanco */
                color: black;            /* Color de texto negro */
            }
        </style>
    </head>

    <body>
        <h1>Diseño Minimalista</h1>
        <p>Este documento ahora tiene un fondo blanco y letras negras desde el CSS del header.</p>
    </body>

</html>

La indentación es importante porque permite que el código sea más ordenado y fácil de leer.

¿Cómo se colocan títulos del 1 al 6 en HTML?

En HTML los títulos se crean utilizando las etiquetas h1 hasta h6.

El h1 es el título más importante y el h6 es el menos importante.

Ejemplo: 

<h1>Título principal</h1>
<h2>Subtítulo</h2>
<h3>Sección</h3>
<h4>Subsección</h4>
<h5>Título menor</h5>
<h6>Título más pequeño</h6>

Para probar este código, debes colocarlo dentro de la etiqueta body de tu archivo HTML, ya que el contenido visible de la página web se escribe dentro de esta sección.

¿Cómo colocar un párrafo en HTML?

Para colocar un párrafo en HTML se utiliza la etiqueta p, la cual permite escribir bloques de texto dentro de una página web.

Ejemplo: 

<p>Este es un párrafo de texto en una página web.</p>

Para probar este código, debes colocarlo dentro de la etiqueta body de tu archivo HTML.

¿Cómo hacer un salto de línea en HTML?

Para hacer un salto de línea en HTML se utiliza la etiqueta br, que permite continuar el texto en la siguiente línea sin crear un nuevo párrafo.

Ejemplo: 

<p>
Primera línea de texto.<br>
Segunda línea de texto.<br>
Tercera línea de texto.
</p>

¿Cómo crear una línea horizontal en HTML?

Para crear una línea horizontal en HTML se utiliza la etiqueta hr, la cual permite separar secciones de contenido dentro de una página web.

Ejemplo: 

<p>Primer contenido</p>

<hr>

<p>Segundo contenido</p>

¿Cómo colocar texto en negrita en HTML?

Para colocar texto en negrita se utiliza la etiqueta b, la cual hace que el texto se vea más destacado.

Ejemplo: 

<p>Este texto está en <b>negrita</b>.</p>

¿Cómo colocar texto en cursiva en HTML?

Para colocar texto en cursiva se utiliza la etiqueta i, la cual inclina el texto para resaltarlo.

Ejemplo: 

<p>Este texto está en <i>cursiva</i>.</p>

¿Cómo colocar listas en HTML?

En HTML se pueden crear listas para organizar información.

Existen principalmente dos tipos de listas: listas ordenadas y listas no ordenadas.

Los elementos de la lista se crean con la etiqueta li.

Lista no ordenada (con viñetas): 

<ul>
  <li>Manzana</li>
  <li>Naranja</li>
  <li>Banana</li>
</ul>

Lista ordenada (con números): 

<ol>
  <li>Primer paso</li>
  <li>Segundo paso</li>
  <li>Tercer paso</li>
</ol>

La lista ordenada permite cambiar el estilo de numeración utilizando el atributo type. 

<ol type="A">
  <li>Primer punto</li>
  <li>Segundo punto</li>
  <li>Tercer punto</li>
</ol>

Tipos más comunes:

  • 1 → números (por defecto)
  • A → letras mayúsculas
  • a → letras minúsculas
  • I → números romanos mayúsculos
  • i → números romanos minúsculos

Se utiliza el atributo start para indicar desde qué número comenzará la lista. 

<ol start="5">
  <li>Paso cinco</li>
  <li>Paso seis</li>
  <li>Paso siete</li>
</ol>

Se utiliza el atributo reversed para mostrar la lista en orden descendente. 

<ol reversed>
  <li>Tercer lugar</li>
  <li>Segundo lugar</li>
  <li>Primer lugar</li>
</ol>

¿Cómo crear enlaces en HTML?

Para crear enlaces en HTML se utiliza la etiqueta a (anchor).

Esta etiqueta permite dirigir al usuario a otra página web, archivo o sección dentro de la misma página.

Ejemplo: 

<a href="https://www.google.com">Ir a Google</a>

Se puede abrir el enlace en una nueva pestaña con el atributo target="_blank" 

<a href="https://www.wikipedia.org" target="_blank">Abrir Wikipedia</a>

¿Cómo agrupar elementos en HTML?

Para agrupar elementos en HTML se utiliza la etiqueta div, la cual permite reunir varios elementos dentro de un mismo contenedor.

Ejemplo: 

<div>
  <h2>Información del curso</h2>
  <p>Este curso enseña desarrollo web.</p>
  <a href="#">Leer más</a>
</div>

¿Cómo colocar imágenes en HTML?

Para mostrar imágenes en una página web se utiliza la etiqueta img.

Existen dos formas comunes de usar imágenes en HTML:

  • Usando una imagen descargada en la computadora
  • Usando la URL de una imagen en internet

Usar una imagen descargada

Primero debes descargar la imagen y guardarla dentro de tu proyecto.

Una forma organizada de trabajar es crear una carpeta llamada img dentro de tu proyecto.

Ejemplo: 

mi-proyecto
│
├── index.html
└── img
    └── paisaje.jpg

Luego puedes mostrar la imagen en tu página HTML indicando la ruta de la carpeta. 

<img src="img/paisaje.jpg" alt="Paisaje de montañas">

Explicación

  • img → etiqueta para mostrar imágenes
  • src → indica la ubicación de la imagen
  • alt → describe la imagen

Usar una imagen desde internet (URL)

También se puede mostrar una imagen utilizando la dirección de internet de la imagen. 

<img src="https://images.unsplash.com/photo-1506744038136-de0b4c0b4c4b" alt="Paisaje natural">

Cambiar el tamaño de la imagen

Puedes modificar el tamaño utilizando el atributo width. 

<img src="img/paisaje.jpg" alt="Paisaje" width="300">

¿Cómo crear tablas en HTML?

Las tablas en HTML permiten organizar información en filas y columnas, de forma similar a una hoja de cálculo.

Para crear una tabla se utilizan tres etiquetas principales:

  • table → define la tabla
  • tr → define una fila
  • td → define una celda

Ejmeplo: 

<table>

  <tr>
    <td>Juan</td>
    <td>20</td>
  </tr>

  <tr>
    <td>María</td>
    <td>22</td>
  </tr>

</table>

Tabla con encabezados

Para los títulos utiliza la etiqueta th.

Ejemplo: 

<table>

  <tr>
    <th>Nombre</th>
    <th>Edad</th>
  </tr>

  <tr>
    <td>Juan</td>
    <td>20</td>
  </tr>

  <tr>
    <td>María</td>
    <td>22</td>
  </tr>

</table>

¿Cómo unir celdas en tablas HTML?

En HTML es posible unir celdas de una tabla para que ocupen más espacio.

Esto se puede hacer horizontalmente o verticalmente.

Para ello se utilizan dos atributos:

  • colspan → une celdas horizontalmente (columnas)
  • rowspan → une celdas verticalmente (filas)

Ejemplo Unir celdas horizontalmente 

<table border="1">

  <tr>
    <th colspan="2">Información del estudiante</th>
  </tr>

  <tr>
    <td>Nombre</td>
    <td>Edad</td>
  </tr>

  <tr>
    <td>Juan</td>
    <td>20</td>
  </tr>

</table>

Ejemplo Unir celdas verticalmente 

<table border="1">

  <tr>
    <th>Nombre</th>
    <th>Materia</th>
  </tr>

  <tr>
    <td rowspan="2">Juan</td>
    <td>Matemática</td>
  </tr>

  <tr>
    <td>Programación</td>
  </tr>

</table>

¿Cómo crear botones en HTML?

Los botones permiten que el usuario realice acciones en una página web, como enviar información, ejecutar funciones o navegar a otra página.

Ejemplo: 

<button>Haz clic aquí</button>

Botón dentro de un enlace

Un botón también puede utilizarse para abrir otra página web. 

<a href="https://www.google.com">
  <button>Ir a Google</button>
</a>

Botón con atributo type

Los botones también pueden tener diferentes tipos. 

<button type="button">Botón normal</button>

Algunos tipos comunes son:

  • button → botón normal
  • submit → enviar formularios
  • reset → reiniciar formularios

¿Cómo crear elementos para guardar datos en HTML?

Para permitir que los usuarios ingresen información en una página web, se utilizan los inputs.

Input de texto

Permite que el usuario escriba información. 

<input type="text">

Input con etiqueta

Cuando se crean inputs, es recomendable usar la etiqueta label.

Para relacionar correctamente la etiqueta con el campo se utilizan los atributos for y id.

Ejemplo: 

<label for="nombre">Nombre</label>
<input type="text" id="nombre">

Explicación

  • label → muestra el texto que describe el campo.
  • for="nombre" → conecta la etiqueta con el input.
  • id="nombre" → identifica el campo de entrada.

Input para correo electrónico

Se utiliza cuando el usuario debe ingresar un correo electrónico.

Ejemplo: 

<label for="correo">Correo electrónico</label>
<input type="email" id="correo">

Input para contraseña

Permite ingresar una contraseña y oculta los caracteres que se escriben.

Ejemplo: 

<label for="clave">Contraseña</label>
<input type="password" id="clave">

Input para números

Se utiliza para ingresar valores numéricos.

Ejemplo: 

<label for="edad">Edad</label>
<input type="number" id="edad">

¿Cómo crear un formulario en HTML?

Un formulario permite que los usuarios ingresen y envíen información en una página web, como nombres, correos o contraseñas.

Ejemplo: 

<form>

<label for="nombre">Nombre</label>
<input type="text" id="nombre">

<br><br>

<label for="correo">Correo electrónico</label>
<input type="email" id="correo">

<br><br>

<label for="clave">Contraseña</label>
<input type="password" id="clave">

<br><br>

<button type="submit">Enviar</button>

</form>

Explicación

  • form → crea el formulario.
  • label → describe el campo que debe llenar el usuario.
  • input → campo donde el usuario escribe información.
  • button → botón para enviar el formulario.

Formularios con GET y POST

Cuando un formulario envía información a un servidor, se utilizan los atributos action y method dentro de la etiqueta form.

  • action → indica a qué página se enviarán los datos.
  • method → indica cómo se enviarán los datos.

Los métodos más usados son:

  • GET → envía los datos en la URL.
  • POST → envía los datos de forma oculta en el cuerpo de la petición

Ejemplo de GET 

<form action="procesar.html" method="get">

<label for="nombre">Nombre</label>
<input type="text" id="nombre" name="nombre">

<br><br>

<label for="correo">Correo</label>
<input type="email" id="correo" name="correo">

<br><br>

<button type="submit">Enviar</button>

</form>

Ejemplo de POST 

<form action="procesar.html" method="post">

<label for="usuario">Usuario</label>
<input type="text" id="usuario" name="usuario">

<br><br>

<label for="clave">Contraseña</label>

En los formularios también es posible permitir que el usuario seleccione una opción de una lista.

¿Cómo seleccionar una opción?

Cuando se necesita que el usuario elija solo una opción entre varias, se utilizan los radio button. Este tipo de control permite seleccionar una única alternativa dentro de un grupo.

Para que los radio button funcionen correctamente, todos deben tener el mismo atributo name. De esta manera el navegador entiende que pertenecen al mismo grupo y solo permitirá seleccionar una opción. 

<form>

<p>Seleccione su turno</p>

<input type="radio" id="manana" name="turno" value="manana">
<label for="manana">Mañana</label>
<br>

<input type="radio" id="tarde" name="turno" value="tarde">
<label for="tarde">Tarde</label>
<br>

<input type="radio" id="noche" name="turno" value="noche">
<label for="noche">Noche</label>

<br><br>

<button type="submit">Enviar</button>

</form>

En este ejemplo se presentan tres opciones: Mañana, Tarde y Noche. Todas las opciones tienen el mismo atributo name="turno", lo que indica que pertenecen al mismo grupo. Por esta razón, el usuario solo podrá seleccionar una sola opción.

El atributo value representa el dato que se enviará cuando el formulario sea enviado. Por ejemplo, si el usuario selecciona Tarde, el formulario enviará el valor tarde.

Seleccionar varias opciones

Cuando se necesita que el usuario pueda elegir varias opciones al mismo tiempo, se utilizan los checkbox. A diferencia de los radio button, los checkbox permiten marcar una o más opciones dentro de un formulario.

Cada opción utiliza type="checkbox". El usuario puede marcar todas las opciones que considere necesarias. 

<form>

<label for="html">HTML</label>
<br>

<label for="css">CSS</label>
<br>

<label for="javascript">JavaScript</label>

<br><br>

<button type="submit">Enviar</button>

En este ejemplo el usuario puede seleccionar uno, dos o todos los lenguajes. Cada opción funciona de manera independiente, por lo que se pueden marcar varias al mismo tiempo.

¿Cómo organizar correctamente una página web para que sea fácil de entender para los buscadores?

Para organizar correctamente una página web y mejorar las buenas prácticas de SEO, HTML utiliza etiquetas semánticas. Estas etiquetas ayudan a estructurar el contenido de la página y permiten que los motores de búsqueda comprendan mejor qué tipo de información contiene cada parte del sitio.

Las etiquetas semánticas más utilizadas son header, nav, main, section, article, aside y footer. Cada una tiene una función específica dentro de la estructura de la página. 

<header>
  <h1>Mi sitio web</h1>
</header>

<nav>
  <a href="#">Inicio</a>
  <a href="#">Servicios</a>
  <a href="#">Contacto</a>
</nav>

<main>

  <section>
    <h2>Servicios</h2>

    <article>
      <h3>Desarrollo web</h3>
      <p>Creamos páginas web modernas.</p>
    </article>

    <article>
      <h3>Aplicaciones móviles</h3>
      <p>Desarrollamos apps para Android.</p>
    </article>

  </section>

  <aside>
    <p>Publicidad o información adicional</p>
  </aside>

</main>

<footer>
  <p>© 2026 Mi sitio web</p>
</footer>

En este ejemplo cada etiqueta cumple una función específica:

header representa el encabezado del sitio web.
nav contiene los enlaces de navegación.
main agrupa el contenido principal de la página.
section divide el contenido en secciones.
article contiene contenido independiente como noticias o artículos.
aside muestra información secundaria o complementaria.
footer representa el pie de página.

Utilizar esta estructura mejora la organización del contenido y ayuda a que los motores de búsqueda comprendan mejor la página.

¿Cómo crear un encabezado en una página web?

Para crear un encabezado en una página web se utiliza la etiqueta header. Esta etiqueta representa la parte superior del sitio y generalmente contiene el título del sitio, un logotipo o información introductoria.

El encabezado ayuda a organizar la página y permite que los usuarios identifiquen rápidamente de qué trata el sitio web. 

<header>

  <h1>Mi sitio web</h1>

  <p>Bienvenido a mi página</p>

</header>

En este ejemplo se crea un encabezado con un título principal y un pequeño texto de bienvenida.

Normalmente el header se coloca al inicio de la página y puede contener también el menú de navegación del sitio.

¿Cómo crear un menú de navegación con nav?

Para crear un menú de navegación en una página web se utiliza la etiqueta nav. Esta etiqueta se emplea para agrupar enlaces que permiten al usuario desplazarse a diferentes páginas o secciones del sitio.

Dentro de la etiqueta nav normalmente se colocan enlaces utilizando la etiqueta a. Cada enlace representa una opción del menú.

En este ejemplo no utilizaremos CSS, por lo tanto el menú aparecerá de forma vertical, mostrando un enlace debajo del otro. 

<nav>

  <a href="#">Inicio</a>
  <br>

  <a href="#">Servicios</a>
  <br>

  <a href="#">Cursos</a>
  <br>

  <a href="#">Contacto</a>

</nav>

En este ejemplo se crea un menú con cuatro enlaces: Inicio, Servicios, Cursos y Contacto. Cada enlace utiliza la etiqueta a y el atributo href, que indica a qué página se dirigirá el usuario cuando haga clic.

La etiqueta nav ayuda a organizar los enlaces del sitio y permite que los buscadores identifiquen fácilmente el menú de navegación.

¿Cómo crear secciones en una página web?

Para dividir el contenido de una página web en diferentes partes se utiliza la etiqueta section. Esta etiqueta permite organizar la información en bloques que tratan un mismo tema.

Cada sección normalmente tiene un título utilizando una etiqueta h2 o h3, seguido de un párrafo o contenido relacionado. Esto ayuda a que la página esté mejor organizada y sea más fácil de entender tanto para los usuarios como para los motores de búsqueda. 

<section>

  <h2>Nuestros servicios</h2>

  <p>
  Ofrecemos desarrollo de páginas web, aplicaciones móviles
  y cursos de programación.
  </p>

</section>

En este ejemplo se crea una sección llamada Nuestros servicios. Dentro de la sección se coloca un título y un párrafo que describe la información de ese apartado.

En una página web pueden existir varias etiquetas section, cada una representando una parte diferente del contenido. 

<section>
  <h2>Servicios</h2>
  <p>Desarrollo web y aplicaciones móviles.</p>
</section>

<section>
  <h2>Cursos</h2>
  <p>Cursos de programación y desarrollo web.</p>
</section>

De esta manera se puede dividir la página en diferentes secciones, permitiendo que el contenido esté más organizado.

¿Cómo crear artículos en una página web?

Para mostrar contenido independiente dentro de una página web se utiliza la etiqueta article. Esta etiqueta se usa para representar información que puede entenderse por sí sola, como una noticia, una publicación de blog o un artículo informativo.

Generalmente un article contiene un título y uno o más párrafos que explican el contenido. 

<article>

  <h3>Curso de programación</h3>

  <p>

  </p>

En este ejemplo se crea un artículo con un título y un párrafo que describe el contenido.

La etiqueta section representa una parte del contenido que trata sobre un mismo tema, mientras que la etiqueta article representa contenido independiente como noticias, publicaciones o información específica. 

<section>

En este ejemplo se crea una sección llamada Cursos disponibles. Dentro de esa sección se agregan tres artículos diferentes, cada uno con su propio título y descripción.

Esta estructura permite organizar el contenido de manera clara y facilita que los buscadores comprendan mejor la información del sitio web.

¿Cómo crear una parte secundaria usando aside?

Cuando una página web tiene información adicional o complementaria al contenido principal, se utiliza la etiqueta aside. Esta etiqueta representa contenido secundario que está relacionado con la página, pero que no forma parte del contenido principal.

Dentro de aside se puede colocar información como publicidad, enlaces relacionados, recomendaciones, noticias o información extra. 

<aside>

  <h3>Información adicional</h3>

  <p>

  </p>

En este ejemplo se crea una sección secundaria con un título y un párrafo que muestra información adicional para el usuario.

La etiqueta aside normalmente se coloca dentro de main junto al contenido principal de la página. 

<main>

  <section>
    <h2>Cursos</h2>

  </section>

  <aside>
    <p>Publicidad o contenido adicional.</p>
  </aside>

De esta forma se separa el contenido principal del contenido secundario, mejorando la organización de la página.

¿Cómo crear el pie de página en una página web?

El pie de página se crea utilizando la etiqueta footer. Esta etiqueta se coloca generalmente al final de la página y contiene información adicional sobre el sitio web.

Dentro del footer se suelen colocar elementos como derechos de autor, información de contacto, enlaces a redes sociales o información del autor de la página. 

<footer>

  <p>© 2026 Mi sitio web</p>

</footer>

En este ejemplo se crea un pie de página con un texto que indica los derechos de autor del sitio web.

El footer normalmente se coloca al final del documento HTML, después del contenido principal de la página. 

<footer>

  <p>© 2026 Mi sitio web</p>
  <p>Contacto: info@misitio.com</p>

</footer>

De esta manera el usuario puede encontrar información adicional del sitio en la parte final de la página.

¿Qué es el CSS?

Es el lenguaje que se utiliza para dar diseño y estilo a las páginas web creadas con HTML.

Mientras HTML estructura el contenido, CSS se encarga de la apariencia visual de la página.

Con CSS se pueden modificar aspectos como:

  • Colores
  • Tipos de letra
  • Tamaños de texto
  • Espacios y márgenes
  • Posición de los elementos

¿Cómo limpiar el CSS que viene por defecto en el navegador?

Los navegadores aplican estilos automáticos a muchos elementos HTML. Por ejemplo, los títulos tienen márgenes, las listas tienen sangrías y los párrafos tienen espacio entre ellos.

Si queremos diseñar una página web desde cero, es recomendable eliminar o reiniciar estos estilos. A esto se le conoce como CSS Reset. De esta forma todos los navegadores comienzan con los mismos estilos básicos.

Una forma sencilla de hacerlo es reiniciando los márgenes y espaciados de todos los elementos utilizando el selector universal *. 

* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

En este ejemplo se eliminan los márgenes y los espacios internos que los navegadores aplican automáticamente.

La propiedad box-sizing: border-box permite que el ancho y alto de los elementos se calculen de una forma más fácil cuando se agregan bordes o padding.

Este pequeño código suele colocarse al inicio del archivo CSS para comenzar a diseñar la página sin los estilos por defecto del navegador.

¿Qué es JS?

JavaScript (JS) es un lenguaje de programación que se utiliza para agregar funcionalidad e interactividad a las páginas web.

Gracias a JavaScript, las páginas web pueden responder a las acciones del usuario, como hacer clic en botones, enviar formularios o mostrar información sin recargar la página.
