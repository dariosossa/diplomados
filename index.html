Evolución del desarrollo web

¿Cómo era la Web 1.0?

La Web 1.0 se caracterizaba por estar formada por páginas web estáticas, donde la información era publicada por los administradores del sitio y los usuarios solo podían leer el contenido sin interactuar con él.

Las páginas estaban compuestas principalmente por texto, imágenes y enlaces, y su función principal era mostrar información, similar a una biblioteca digital.

¿Cómo era la Web 2.0?

La Web 2.0 se caracterizó por permitir la interacción y participación activa de los usuarios en internet.

A diferencia de la Web 1.0, donde los usuarios solo podían leer información, en la Web 2.0 las personas pueden crear, compartir y comentar contenido en diferentes plataformas digitales.

En esta etapa surgieron servicios como redes sociales, blogs, wikis, plataformas de video y aplicaciones web interactivas, donde los usuarios se convierten también en creadores de contenido.

Ejemplos claros de Web 2.0 son plataformas como Facebook, YouTube y Twitter, donde la información es generada y compartida por millones de usuarios.

¿Cómo es la actual Web 3.0?

La Web 3.0 se caracteriza por ser una web más inteligente, conectada y personalizada, donde las aplicaciones pueden analizar grandes cantidades de información para ofrecer resultados más precisos a los usuarios.

En esta etapa de internet se utilizan tecnologías como inteligencia artificial, procesamiento de datos, computación en la nube y sistemas semánticos, que permiten que las plataformas comprendan mejor la información y el comportamiento de las personas.

Evolución del desarrollo móvil

¿Cómo fue la primera etapa?

La primera etapa del desarrollo móvil se caracterizó por el uso de teléfonos móviles básicos que tenían funciones limitadas, principalmente llamadas telefónicas y envío de mensajes de texto (SMS).

En ese periodo, los dispositivos tenían pantallas pequeñas, poca capacidad de almacenamiento y recursos de procesamiento muy reducidos, por lo que las aplicaciones eran simples y con pocas funciones.

¿Cómo fue la segunda etapa?

La segunda etapa del desarrollo móvil se caracterizó por la aparición de los primeros teléfonos inteligentes o smartphones, que comenzaron a ofrecer más funciones además de llamadas y mensajes.

En esta etapa los dispositivos incorporaron pantallas más grandes, conexión a internet, cámaras y sistemas operativos más avanzados, lo que permitió el desarrollo de aplicaciones más complejas.

También surgieron las primeras tiendas de aplicaciones, donde los usuarios podían descargar diferentes programas para ampliar las funciones de sus dispositivos.

¿Cómo es la etapa actual?

La etapa actual del desarrollo móvil se caracteriza por el uso de teléfonos inteligentes avanzados con gran capacidad de procesamiento, conexión permanente a internet y aplicaciones cada vez más complejas.

En esta etapa los dispositivos móviles permiten realizar múltiples actividades como comunicación, comercio electrónico, educación en línea, redes sociales, entretenimiento y trabajo remoto.

Además, las aplicaciones móviles actuales integran tecnologías como inteligencia artificial, computación en la nube, geolocalización y servicios digitales en tiempo real, lo que permite ofrecer experiencias más rápidas y personalizadas a los usuarios.

¿Qué es la estructura básica de HTML?

La estructura básica de HTML es la organización inicial que debe tener toda página web para que el navegador pueda interpretarla correctamente.

Esta estructura define el tipo de documento, la configuración de la página y el contenido que se mostrará al usuario.

Ejemplo de estructura básica:

<!DOCTYPE html>
<html>

    <head>

    </head>

    <body>

    </body>

</html>

La indentación es importante porque permite que el código sea más ordenado y fácil de leer.

¿Cómo se colocan títulos del 1 al 6 en HTML?

En HTML los títulos se crean utilizando las etiquetas h1 hasta h6.

El h1 es el título más importante y el h6 es el menos importante.

Ejemplo: 

<h1>Título principal</h1>
<h2>Subtítulo</h2>
<h3>Sección</h3>
<h4>Subsección</h4>
<h5>Título menor</h5>
<h6>Título más pequeño</h6>

Para probar este código, debes colocarlo dentro de la etiqueta body de tu archivo HTML, ya que el contenido visible de la página web se escribe dentro de esta sección.

¿Cómo colocar un párrafo en HTML?

Para colocar un párrafo en HTML se utiliza la etiqueta p, la cual permite escribir bloques de texto dentro de una página web.

Ejemplo: 

<p>Este es un párrafo de texto en una página web.</p>

Para probar este código, debes colocarlo dentro de la etiqueta body de tu archivo HTML.

¿Cómo hacer un salto de línea en HTML?

Para hacer un salto de línea en HTML se utiliza la etiqueta br, que permite continuar el texto en la siguiente línea sin crear un nuevo párrafo.

Ejemplo: 

<p>
Primera línea de texto.<br>
Segunda línea de texto.<br>
Tercera línea de texto.
</p>

¿Cómo crear una línea horizontal en HTML?

Para crear una línea horizontal en HTML se utiliza la etiqueta hr, la cual permite separar secciones de contenido dentro de una página web.

Ejemplo:

Tarija: Tierra de Gigantes Prehistóricos

Un Viaje al Pleistoceno en el Valle

Hace millones de años, la cuenca de Tarija no era el valle seco que conocemos hoy, sino un ecosistema vibrante de humedales y praderas que servía de hogar a una megafauna impresionante. Debido a procesos de sedimentación únicos, los fósiles encontrados en esta región son de los mejores conservados en toda Sudamérica. Al caminar por zonas como el barrio de San Roque o las laderas de los cerros circundantes, los geólogos han descubierto depósitos que actúan como una cápsula del tiempo, permitiéndonos reconstruir cómo era la vida mucho antes de la aparición del ser humano moderno.

El Mastodonte y el Megaterio

Entre las criaturas más fascinantes que habitaron Tarija destaca el Mastodonte (Cuvieronius), un pariente lejano de los elefantes modernos que poseía colmillos enormes y ligeramente curvados. Junto a él, el Megaterio, un perezoso terrestre del tamaño de un elefante, dominaba las llanuras alimentándose de las copas de los árboles. Estos gigantes pesaban varias toneladas y su estructura ósea revela adaptaciones sorprendentes a los cambios climáticos de la época. Los hallazgos de molares y defensas en las excavaciones locales han permitido a los científicos determinar que estos animales convivieron en un equilibrio ecológico perfecto durante milenios.

La Importancia del Museo Paleontológico

El Museo Nacional Paleontológico y Arqueológico de Tarija resguarda una colección invaluable que atrae a investigadores de todo el globo. Más allá de la exhibición de esqueletos completos, el museo cumple una función crítica en la preservación del patrimonio nacional. La erosión natural y la expansión urbana son amenazas constantes para los yacimientos fósiles que aún permanecen enterrados. Es imperativo fomentar una cultura de respeto y reporte de hallazgos, ya que cada fragmento de hueso fosilizado es una pieza del rompecabezas que nos cuenta la historia de la evolución biológica en nuestro continente y los efectos de las glaciaciones pasadas.

Arquitectura y Evolución

El Trazado Colonial y la Estética Republicana

El centro histórico de Tarija es un testimonio viviente de la transición entre el orden colonial español y la elegancia de la época republicana. Originalmente diseñada bajo el modelo de "damero" o cuadrícula, la ciudad creció alrededor de su plaza principal, donde las familias más influyentes construyeron casonas de adobe con muros de casi un metro de espesor. Estas estructuras no solo eran un símbolo de estatus, sino una respuesta inteligente al clima del valle, manteniendo el interior fresco durante los veranos intensos y conservando el calor durante las noches de invierno. La preservación de estas fachadas es hoy un reto arquitectónico constante.

La Casa Dorada: Un Ícono de la Modernidad

Sin duda, la joya de la corona arquitectónica es la Casa Dorada, cuya construcción finalizó a principios del siglo XX. Con un estilo ecléctico que mezcla influencias victorianas y neoclásicas, su fachada destaca por las estatuas que coronan el tejado y sus detalles en pan de oro. El interior es un viaje en el tiempo: espejos de cristal de roca traídos de Venecia, tapices franceses y techos pintados a mano por artistas europeos. Representa una era en la que Tarija miraba hacia el exterior, integrando modas internacionales en el corazón de su cultura local, convirtiéndose en el centro cultural más importante del sur de Bolivia.

Los Balcones y Zaguanes como Espacios Sociales

Un elemento distintivo de las viviendas tradicionales tarijeñas es el uso del zaguán y los balcones de madera tallada. El zaguán funcionaba como una zona de transición entre la vida pública de la calle y la privacidad del patio central lleno de jazmines y malvones. Los balcones, por su parte, permitían a los habitantes observar el ritmo de la ciudad sin salir de casa, fomentando una cultura de vecindad y vigilancia comunitaria que aún se respira en las calles empedradas. Estos detalles ornamentales, aunque frágiles ante el paso del tiempo, son la firma visual de una identidad que se niega a desaparecer frente al avance del hormigón moderno.

El Telescopio Webb y el Amanecer Cósmico

Una Ventana al Pasado de 13 Mil Millones de Años

Mirar a través del telescopio James Webb es, literalmente, mirar hacia atrás en el tiempo. Situado en el punto de equilibrio Lagrange L2, a un millón y medio de kilómetros de la Tierra, este ingenio tecnológico utiliza espejos bañados en oro para captar la luz infrarroja de las primeras estrellas que se encendieron tras el Big Bang. A diferencia del Hubble, que observaba principalmente luz visible, el Webb puede atravesar las densas nubes de polvo interestelar, revelando viveros de estrellas y sistemas planetarios en formación que antes eran invisibles para la humanidad. Cada imagen enviada es una pieza nueva en el mapa de nuestra propia existencia.

La Búsqueda de Mundos Habitables

Uno de los objetivos más ambiciosos de esta misión es el análisis de las atmósferas de exoplanetas, mundos que orbitan otras estrellas lejos de nuestro sistema solar. Mediante una técnica llamada espectroscopía, los científicos pueden detectar la presencia de vapor de agua, dióxido de carbono y quizás señales biológicas como el metano. Si encontráramos un planeta con una composición atmosférica similar a la Tierra, cambiaría nuestra comprensión del universo para siempre. El Webb ya ha analizado el sistema TRAPPIST-1, encontrando detalles fascinantes que nos acercan un paso más a responder la pregunta eterna: ¿estamos solos en el cosmos?

Ingeniería en el Vacío Absoluto

La construcción y despliegue del Webb se considera la hazaña de ingeniería más compleja de la historia espacial. El telescopio tuvo que ser lanzado plegado dentro de un cohete y desplegarse en el espacio como un origami de alta tecnología. Su parasol, del tamaño de una cancha de tenis, lo protege del calor del Sol, permitiendo que sus instrumentos operen a temperaturas cercanas al cero absoluto. Cualquier fallo en los cientos de puntos de liberación habría significado el fin de la misión. Hoy, su éxito es un testimonio de lo que la cooperación internacional entre agencias de Estados Unidos, Europa y Canadá puede lograr cuando se busca expandir los límites del conocimiento.

Abismos Marinos: La Última Frontera

Explorando la Zona Hadal

A pesar de que cubren más del 70% de la superficie terrestre, conocemos menos sobre el fondo del mar que sobre la superficie de la Luna. La Zona Hadal, que comprende las fosas más profundas del océano por debajo de los 6.000 metros, representa un entorno de condiciones extremas donde la presión es mil veces superior a la del nivel del mar. En este reino de oscuridad perpetua, la vida no depende de la fotosíntesis, sino de procesos químicos fascinantes que desafían nuestra comprensión biológica tradicional.

Adaptaciones Biológicas y Quimiosíntesis

Las criaturas que habitan estas profundidades, como el Pez Caracol o los anfípodos gigantes, han desarrollado cuerpos translúcidos y carentes de pigmentación. Al no existir luz solar, el ecosistema se sustenta gracias a las fuentes hidrotermales: grietas en el lecho marino que expulsan minerales calientes. Aquí, las bacterias realizan la quimiosíntesis, convirtiendo sustancias tóxicas como el sulfuro de hidrógeno en energía vital. Este proceso sugiere que la vida podría haber surgido en estas chimeneas submarinas hace miles de millones de años.

El Impacto Humano en el Abismo

Resulta alarmante descubrir que, incluso en la Fosa de las Marianas (el punto más profundo del planeta), se han encontrado rastros de microplásticos y contaminantes químicos. La conservación de estos ecosistemas es crucial, no solo por su biodiversidad única, sino porque actúan como reguladores térmicos del planeta. Estudiar estos organismos extremófilos podría darnos la clave para desarrollar nuevos antibióticos y entender cómo podría sobrevivir la vida en océanos extraterrestres, como los de la luna Europa de Júpiter.

Música, Sordera y Libertad

El Silencio que Cambió la Historia del Arte

La figura de Ludwig van Beethoven marcó la transición definitiva entre el Clasicismo y el Romanticismo. A sus 26 años, el compositor empezó a notar los primeros síntomas de una pérdida auditiva que terminaría en una sordera total. Sin embargo, este aislamiento acústico no detuvo su genio; al contrario, lo obligó a escuchar la música en su mente con una pureza técnica inigualable. Sus últimas obras no fueron escritas para el oído humano de la época, sino para la eternidad, desafiando todas las reglas armónicas establecidas hasta entonces.

La Novena Sinfonía: El Himno de la Humanidad

En 1824, Beethoven estrenó su Novena Sinfonía, la primera en la historia en incluir un coro y solistas en una estructura sinfónica. El texto, basado en la Oda a la Alegría de Friedrich Schiller, es un llamado a la hermandad universal. Se dice que al terminar el estreno, el compositor —completamente sordo— tuvo que ser girado por los músicos para que pudiera ver la ovación de pie del público, ya que no podía escuchar ni el estruendo de los aplausos ni el de su propia orquesta.

Legado en la Era de la Reproducción Digital

Hoy en día, la estructura de la Novena sigue siendo un estándar técnico. Un dato curioso es que la capacidad original de los Compact Discs (74 minutos) fue diseñada específicamente para que la sinfónica completa pudiera caber en un solo disco sin interrupciones. La música de Beethoven sigue recordándonos que el arte es una victoria de la voluntad humana sobre la adversidad física. Su influencia se extiende desde las salas de concierto más prestigiosas hasta las bandas sonoras del cine contemporáneo, manteniendo una vigencia absoluta en pleno siglo XXI.

¿Cómo colocar listas en HTML?

En HTML se pueden crear listas para organizar información.

Existen principalmente dos tipos de listas: listas ordenadas y listas no ordenadas.

Los elementos de la lista se crean con la etiqueta li.

Lista no ordenada (con viñetas): 

<ul>
  <li>Manzana</li>
  <li>Naranja</li>
  <li>Banana</li>
</ul>

Lista ordenada (con números): 

<ol>
  <li>Primer paso</li>
  <li>Segundo paso</li>
  <li>Tercer paso</li>
</ol>

La lista ordenada permite cambiar el estilo de numeración utilizando el atributo type. 

<ol type="A">
  <li>Primer punto</li>
  <li>Segundo punto</li>
  <li>Tercer punto</li>
</ol>

Tipos más comunes:

  • 1 → números (por defecto)
  • A → letras mayúsculas
  • a → letras minúsculas
  • I → números romanos mayúsculos
  • i → números romanos minúsculos

Se utiliza el atributo start para indicar desde qué número comenzará la lista. 

<ol start="5">
  <li>Paso cinco</li>
  <li>Paso seis</li>
  <li>Paso siete</li>
</ol>

Se utiliza el atributo reversed para mostrar la lista en orden descendente. 

<ol reversed>
  <li>Tercer lugar</li>
  <li>Segundo lugar</li>
  <li>Primer lugar</li>
</ol>

La Era de las Máquinas y el Vapor

Del Taller Artesanal a la Fábrica Moderna

La Revolución Industrial, iniciada en Gran Bretaña a finales del siglo XVIII, no fue solo un cambio tecnológico, sino una transformación total de la existencia humana. La invención de la máquina de vapor por James Watt permitió mecanizar procesos que antes dependían de la fuerza animal o humana. Este cambio alteró para siempre la demografía global, impulsando una migración masiva del campo a las ciudades en busca de nuevas oportunidades en las nacientes metrópolis industriales.

Innovaciones Técnicas de la Primera Etapa

El despegue industrial se apoyó en tres pilares fundamentales que permitieron un crecimiento económico sin precedentes:

  • El Carbón: La fuente de energía primaria que alimentaba las calderas.
  • El Hierro: Material esencial para la construcción de maquinaria y ferrocarriles.
  • La Industria Textil: El primer sector en adoptar la producción en masa.

Evolución de los Medios de Transporte

La necesidad de mover mercancías rápidamente dio lugar a una revolución en la ingeniería civil:

  1. Construcción de canales navegables para el transporte de carbón a bajo costo.
  2. Desarrollo de la Locomotora de Vapor (The Rocket) por George Stephenson.
  3. Aparición de los primeros barcos de vapor transatlánticos.

Consecuencias Sociales a Largo Plazo

No todo fue progreso técnico; la sociedad tuvo que adaptarse a nuevas y a veces duras realidades laborales:

  1. Surgimiento del proletariado urbano y los sindicatos.
  2. Regulación de las jornadas de trabajo y prohibición del trabajo infantil.
  3. Crecimiento exponencial de la clase media comercial.

Hitos de la Segunda Revolución Industrial

Hacia finales del siglo XIX, la electricidad y el petróleo tomaron el relevo:

  1. Invención de la bombilla incandescente por Thomas Edison.
  2. Producción en cadena implementada por Henry Ford en el sector automotriz.
  3. Desarrollo de la industria química y los primeros plásticos sintéticos.

El Cerebro: La Frontera Final del Conocimiento

La Red de Conexiones más Compleja del Universo

El cerebro humano pesa apenas 1.4 kilogramos, pero contiene aproximadamente 86.000 millones de neuronas. Cada una de estas células se conecta con miles de otras a través de impulsos electroquímicos, creando una red de información cuya complejidad supera a cualquier supercomputadora actual. La neuroplasticidad es la capacidad de este órgano para reorganizarse a sí mismo, creando nuevas rutas de aprendizaje a lo largo de toda la vida del individuo.

Estructura y Funciones de los Lóbulos Cerebrales

Para facilitar su estudio, la corteza cerebral se divide en cuatro áreas principales, cada una responsable de funciones específicas que nos permiten interactuar con el mundo:

  1. Lóbulo Frontal: Sede del razonamiento, la planificación y el control emocional.
  2. Lóbulo Parietal: Encargado de procesar información sensorial como el tacto y el dolor.
  3. Lóbulo Temporal: Fundamental para la memoria y el procesamiento del lenguaje.
  4. Lóbulo Occipital: Dedicado casi exclusivamente a la interpretación de la visión.

El Proceso de la Sinapsis

La comunicación entre neuronas ocurre en un espacio microscópico llamado espacio sináptico. El proceso sigue una secuencia lógica:

  1. Llegada del impulso eléctrico al terminal de la neurona emisora.
  2. Liberación de neurotransmisores (como la dopamina o serotonina).
  3. Recepción de las moléculas por los receptores de la neurona receptora.
  4. Generación de un nuevo impulso eléctrico en la célula destino.

Hábitos para un Cerebro Saludable

La ciencia moderna ha demostrado que el estilo de vida influye directamente en la salud cognitiva a largo plazo. Se recomiendan los siguientes puntos:

  • Actividad Física: Mejora la oxigenación y la producción de factores neurotróficos.
  • Sueño Reparador: Durante el descanso, el cerebro elimina toxinas metabólicas.
  • Aprendizaje de Idiomas: Aumenta la densidad de la materia gris y retrasa el declive cognitivo.

Curiosidades de la Neurociencia

Existen datos que aún hoy sorprenden a los investigadores más experimentados:

  1. El cerebro consume el 20% de la energía total del cuerpo humano.
  2. No existen receptores de dolor en el tejido cerebral (por eso las cirugías pueden ser despiertos).
  3. La información viaja por los nervios a velocidades de hasta 400 km/h.

TOMO I: EL HORIZONTE ROJO

La Expansión hacia las Estrellas

En este primer volumen, analizamos la ambición del ser humano por trascender los límites de la atmósfera terrestre. La misión a Marte no es solo un logro técnico, sino la respuesta a nuestra naturaleza exploradora. Para convertirnos en una especie multiplanetaria, debemos dominar entornos donde la presión atmosférica es mínima y el oxígeno es un recurso manufacturado. La logística de un viaje de esta magnitud requiere una precisión sin precedentes, considerando que una ventana de lanzamiento solo se abre cada 26 meses, cuando la alineación planetaria es óptima.

La supervivencia en la superficie marciana depende de nuestra capacidad para transformar un desierto gélido y radiactivo en un hogar habitable. El uso de materiales locales, como el regolito marciano para fabricar ladrillos mediante impresión 3D, es fundamental para reducir la carga de los cohetes que salen de la Tierra. No se trata solo de llegar, sino de establecer una presencia permanente que garantice la continuidad de nuestra civilización ante posibles cataclismos terrestres.

Desafíos Críticos del Viaje Espacial

El cuerpo humano no evolucionó para vivir en microgravedad. Durante el tránsito y la estancia, los astronautas enfrentan peligros biológicos que la ciencia actual intenta mitigar mediante fármacos y ejercicio intensivo:

  1. Atrofia Muscular y Ósea: La falta de carga gravitatoria provoca una pérdida de densidad ósea similar a la osteoporosis acelerada, lo que requiere trajes de compresión y rutinas de resistencia diarias.
  2. Escudo de Radiación: El espacio profundo está bombardeado por rayos cósmicos galácticos que dañan el ADN. Los hábitats deben estar enterrados o cubiertos por capas de agua para servir de escudo.
  3. Sistemas de Reciclaje Cerrado: En Marte, nada se desperdicia. El sudor, la orina y la humedad ambiental deben purificarse con una eficiencia superior al 98% para mantener el suministro de agua.

El Legado de la Exploración

Este esfuerzo internacional, liderado por agencias gubernamentales y empresas privadas, representa el mayor hito de la ingeniería contemporánea. Las lecciones aprendidas en la Luna con el programa Artemis son los cimientos sobre los que se construye esta nueva era. La humanidad está a punto de dejar de ser una civilización de un solo planeta para convertirse en ciudadana del sistema solar.

TOMO II: EL CÓDIGO DE LA VIDA

La Transformación de la Medicina

Mientras el Tomo I explora las fronteras exteriores, este segundo volumen se adentra en el espacio interior: la biología humana potenciada por la Inteligencia Artificial. La medicina del 2026 ha dejado de ser reactiva para convertirse en una disciplina puramente predictiva. El análisis masivo de bio-datos permite hoy simular el comportamiento de enfermedades en gemelos digitales antes de aplicar cualquier tratamiento en el paciente real.

La integración de la nanotecnología en el torrente sanguíneo ha permitido que el diagnóstico sea constante. Sensores microscópicos monitorizan niveles de glucosa, cortisol y marcadores tumorales en tiempo real, enviando alertas directamente al sistema de salud preventivo. Ya no esperamos a sentir dolor para ir al médico; la intervención ocurre a nivel molecular mucho antes de que el síntoma se manifieste físicamente.

Pilares de la Salud Digital

El nuevo ecosistema sanitario se fundamenta en tecnologías que hace solo una década parecían ciencia ficción, transformando radicalmente la esperanza y calidad de vida:

  1. Análisis Genómico Personalizado: El mapeo completo del ADN de cada individuo permite diseñar dietas, ejercicios y medicamentos específicos que maximizan la eficacia y eliminan efectos secundarios.
  2. Interfaces Cerebro-Máquina: Chips neuronales de alta fidelidad permiten a personas con lesiones medulares controlar prótesis robóticas con el pensamiento, devolviendo la autonomía y el tacto digital.
  3. Edición Genética CRISPR: La capacidad de corregir errores en el código genético directamente en células adultas para curar enfermedades hereditarias que antes se consideraban crónicas.

El Futuro de la Longevidad

Estamos alcanzando un punto de inflexión donde el envejecimiento comienza a tratarse como una patología reversible. La reprogramación celular y la bioimpresión de órganos en 3D —utilizando las propias células del paciente para evitar rechazos— sugieren un futuro donde la sustitución de tejidos dañados sea un procedimiento de rutina. El desafío del mañana no será solo vivir más años, sino asegurar que esos años se vivan con plenitud física y mental.

LA TRANSICIÓN ENERGÉTICA GLOBAL

El Desafío de Reemplazar los Combustibles Fósiles

La humanidad se encuentra en un punto de inflexión crítico. La dependencia histórica del carbón y el petróleo ha impulsado el desarrollo industrial, pero a un costo ambiental insostenible. La descarbonización de la economía no es solo un objetivo ecológico, sino una necesidad de seguridad global. El reto reside en desarrollar fuentes de energía que sean tanto limpias como capaces de suministrar la enorme demanda de las megaciudades modernas de manera constante y confiable.

A diferencia de las energías intermitentes como la solar o la eólica, que dependen de las condiciones climáticas, la ciencia está buscando el "Santo Grial" de la energía: la fusión nuclear. Al imitar el proceso que ocurre en el corazón del Sol, podríamos obtener una fuente de energía virtualmente inagotable, sin emisiones de carbono y con una generación mínima de residuos en comparación con la fisión nuclear tradicional.

Fuentes de Energía en Desarrollo

La matriz energética del futuro será un ecosistema híbrido compuesto por diversas tecnologías complementarias:

  • Energía Fotovoltaica de Perovskita: Una nueva generación de celdas solares más baratas y eficientes que el silicio tradicional.
  • Hidrógeno Verde: Utilizado como vector energético para descarbonizar industrias pesadas y el transporte marítimo.
  • Energía Undimotriz: Captura de la energía cinética de las olas del mar para generar electricidad constante.

Ventajas de la Energía Limpia

La transición hacia un modelo sostenible ofrece beneficios que van más allá de la reducción de gases de efecto invernadero:

  1. Independencia energética para naciones que no poseen yacimientos de hidrocarburos.
  2. Reducción drástica de enfermedades respiratorias causadas por la contaminación urbana.
  3. Estabilización de los costos de energía a largo plazo al no depender de mercados de crudo volátiles.

El Papel del Almacenamiento

El mayor obstáculo actual no es la generación, sino el almacenamiento. El desarrollo de baterías de estado sólido y sistemas de almacenamiento térmico en sales fundidas es esencial para garantizar que la energía capturada durante el día esté disponible durante las horas de mayor consumo nocturno.

MÉTODO POST: DATOS ENCAPSULADOS

Privacidad y Capacidad

A diferencia de GET, el método POST no altera la URL del navegador. Los datos viajan de forma interna en el cuerpo de la petición (HTTP Body). Es como enviar una carta dentro de un sobre sellado: nadie puede ver el contenido desde afuera. Este método es obligatorio para crear cuentas, realizar pagos, subir archivos pesados o enviar contraseñas, ya que no tiene límites estrictos de tamaño y ofrece una capa básica de discreción.

¿Cómo se comportan los elementos en CSS?

Los elementos de una página web se comportan como cajas.

Cada elemento HTML ocupa un espacio rectangular dentro de la página y ese espacio puede modificarse utilizando propiedades como margin, border y padding. Este comportamiento se conoce como Modelo de Caja (Box Model).

Según este modelo, cada elemento está formado por cuatro partes:

MARGIN
BORDER
PADDING
CONTENT
(Text / Image)
Margin
Border
Padding
Content

Para entender cómo funciona el modelo de caja, crearemos un título h1 y luego utilizaremos el inspector de elementos del navegador.

¿Cómo modificar las propiedades?

Podemos modificar las propiedades del modelo de caja utilizando CSS. 

h1 {
  margin: 40px;
  padding: 20px;
  border: 3px solid black;
}

¿Qué podemos hacer con el border?

La propiedad border permite crear un borde alrededor de un elemento. Este borde se dibuja entre el padding y el margin.

Con border podemos modificar principalmente tres aspectos:

  • border-width → grosor del borde
  • border-style → tipo de línea
  • border-color → color del borde

Ejemplo básico

h1 {
  border: 3px solid black;
}

En este ejemplo:

  • 3px → grosor del borde
  • solid → línea continua
  • black → color negro

Tipos de borde

CSS permite usar diferentes estilos de borde. 

h1 {
  border: 4px dashed blue;
}

Algunos tipos de borde son:

  • solid → línea continua
  • dashed (dásht) → línea discontinua
  • dotted → línea de puntos
  • double (dóubl) → doble línea

¿Es posible aplicar el border solo en un lado?

Sí. CSS permite aplicar el borde en un solo lado del elemento.

Podemos usar las siguientes propiedades:

  • border-top → borde superior
  • border-right → borde derecho
  • border-bottom → borde inferior
  • border-left → borde izquierdo

Ejemplo

h1 {
  border-bottom: 4px solid red;
}

En este ejemplo el borde solo aparece en la parte inferior del título.

Esto es muy utilizado para crear separadores o líneas decorativas debajo de los títulos.

¿Cómo redondear las esquinas de un borde?

CSS permite redondear las esquinas de un elemento utilizando la propiedad border-radius.

Esta propiedad se usa mucho en diseño web moderno para crear botones, tarjetas y contenedores con esquinas suaves.

Ejemplo básico

h1 {
  border: 3px solid black;
  border-radius: 10px;
}

En este ejemplo el borde del título tendrá esquinas redondeadas.

Crear un círculo

Si el valor de border-radius es muy grande, el elemento puede volverse circular. 

div {
  width: 150px;
  height: 150px;
  border: 3px solid blue;
  border-radius: 50%;
}

Esto es muy utilizado para crear imágenes de perfil o elementos circulares en una página web.

¿Cómo cambiar el tamaño de un elemento en CSS?

CSS permite modificar el tamaño de los elementos utilizando las propiedades width y height.

  • width → ancho del elemento
  • height → alto del elemento

Ejemplo

div {
  width: 300px;
  height: 150px;
  border: 3px solid black;
}

En este ejemplo el elemento tendrá un ancho de 300 píxeles y una altura de 150 píxeles.

Estas propiedades se utilizan para controlar el tamaño de cajas, contenedores, imágenes y otros elementos de una página web.

¿Qué unidades se pueden usar en CSS?

Cuando modificamos tamaños en CSS, como el ancho, el alto o el tamaño del texto, debemos indicar una unidad de medida.

Las unidades indican cuánto espacio ocupará un elemento en la página.

Las unidades más comunes al comenzar a trabajar con CSS son:

  • px → píxeles
  • % → porcentaje

Unidad px

La unidad px representa píxeles. Es una medida fija y muy utilizada para controlar el tamaño de los elementos con precisión. 

div {
  width: 300px;
  height: 150px;
}

En este ejemplo el elemento tendrá un ancho de 300 píxeles y una altura de 150 píxeles.

Unidad %

La unidad % representa un porcentaje respecto al tamaño del elemento contenedor. 

div {
  width: 50%;
}

Esto significa que el elemento ocupará el 50% del ancho del contenedor donde se encuentra.

Unidad em

La unidad em se basa en el tamaño de letra del elemento padre.

Por defecto, la mayoría de los navegadores usan un tamaño de letra de 16px.

Entonces, los valores se calculan así:

  • 1em = 16px
  • 2em = 32px
  • 0.5em = 8px
p {
  font-size: 2em;
}

Si el tamaño base es 16px, entonces 2em = 32px.

Unidad rem

La unidad rem significa root em.

En este caso el cálculo se basa en el tamaño del elemento raíz de la página, que es el elemento html.

Por defecto también suele ser 16px.

  • 1rem = 16px
  • 2rem = 32px
  • 0.5rem = 8px
p {
  font-size: 1.5rem;
}

Si el tamaño base es 16px, entonces 1.5rem = 24px.

Unidad vw

La unidad vw significa viewport width.

Representa un porcentaje del ancho de la pantalla.

Por ejemplo, si la pantalla mide 1000px de ancho:

  • 100vw = 1000px
  • 50vw = 500px
  • 25vw = 250px
div {
  width: 50vw;
}

El elemento ocupará el 50% del ancho de la pantalla.

Unidad vh

La unidad vh significa viewport height.

Representa un porcentaje de la altura de la pantalla.

Por ejemplo, si la pantalla mide 800px de alto:

  • 100vh = 800px
  • 50vh = 400px
  • 25vh = 200px
div {
  height: 50vh;
}

El elemento ocupará el 50% de la altura de la pantalla.

¿Qué colores se pueden usar en CSS?

En CSS se pueden usar diferentes formas para definir colores.

Las más comunes son:

  • Nombre del color
  • RGB
  • HEX

Colores por nombre

CSS reconoce muchos colores por su nombre. 

h1 {
  color: red;
}

p {
  color: blue;
}

div {
  background-color: yellow;
}

Algunos colores que se pueden usar son:

  • red
  • blue
  • green
  • yellow
  • black
  • white
  • gray
  • orange
  • purple

Colores RGB

RGB significa Red, Green, Blue.

Cada color se define usando valores entre 0 y 255. 

h1 {
  color: rgb(255, 0, 0);
}

Esto significa:

  • 255 rojo
  • 0 verde
  • 0 azul

El resultado es color rojo.

Colores HEX

HEX es otra forma de representar colores usando números y letras. 

h1 {
  color: #ff0000;
}

Este valor también representa el color rojo.

Los colores HEX siempre comienzan con el símbolo #.

Colores RGBA

RGBA es similar a RGB, pero incluye un cuarto valor que controla la transparencia del color.

RGBA significa:

  • Red
  • Green
  • Blue
  • Alpha (transparencia)

El valor alpha va de 0 a 1.

  • 1 → completamente visible
  • 0.5 → semitransparente
  • 0 → completamente transparente
div {
  background-color: rgba(255, 0, 0, 0.5);
}

Este ejemplo crea un fondo rojo con un 50% de transparencia.

¿Cómo se alinean los textos en CSS?

En CSS podemos alinear el texto utilizando la propiedad text-align.

Esta propiedad permite controlar cómo se distribuye el texto dentro de un elemento.

Valores más utilizados

  • left → alinea el texto a la izquierda
  • center → centra el texto
  • right → alinea el texto a la derecha
  • justify → ajusta el texto para que ocupe todo el ancho

h1 {

En este ejemplo el título aparecerá centrado.

Otro ejemplo

p {
  text-align: justify;
}

Esto hará que el texto del párrafo se distribuya de manera uniforme ocupando todo el ancho del elemento.

¿Cómo cambiar el tamaño del texto en CSS?

h1 {

p {

Esto hará que todos los párrafos tengan un tamaño de 18 píxeles.

La propiedad font-size se utiliza para controlar el tamaño de la letra de cualquier elemento de la página.

¿Cómo colocar texto en negrita usando CSS?

p {

Usando valores numéricos

h1 {

La propiedad font-style permite cambiar la forma del texto para resaltar palabras o frases dentro del contenido.

¿Cómo subrayar texto usando CSS?

En CSS podemos subrayar el texto utilizando la propiedad text-decoration.

p {

}

En este ejemplo todos los párrafos aparecerán subrayados.

Otros valores de text-decoration

  • underline → subraya el texto
  • overline → coloca una línea encima del texto
  • line-through → coloca una línea en medio del texto
  • none → elimina cualquier decoración del texto

Ejemplo con línea en medio

p {

Esto mostrará el texto con una línea atravesándolo.

¿Podemos cambiar la forma en que se muestran las letras?

Sí. En CSS podemos modificar cómo aparecen las letras utilizando la propiedad text-transform.

Esta propiedad permite convertir el texto en mayúsculas o minúsculas automáticamente.

Valores más utilizados

  • uppercase → convierte todo el texto en mayúsculas
  • lowercase → convierte todo el texto en minúsculas
  • capitalize → la primera letra de cada palabra en mayúscula

Ejemplo

p {
  text-transform: uppercase;
}

En este ejemplo todo el texto del párrafo se mostrará en mayúsculas.

Otro ejemplo

p {
  text-transform: capitalize;
}

Esto hará que la primera letra de cada palabra aparezca en mayúscula.

¿Cómo cambiar el tipo de letra en CSS?

En CSS podemos cambiar el tipo de letra utilizando la propiedad font-family.

Esta propiedad permite elegir qué fuente se usará para mostrar el texto.

Ejemplo

p {
  font-family: Arial;
}

En este ejemplo todos los párrafos usarán la fuente Arial.

Usar varias fuentes

Es recomendable indicar varias fuentes por si alguna no está disponible en el navegador del usuario. 

p {
  font-family: Arial, Helvetica, sans-serif;
}

El navegador intentará usar la primera fuente. Si no está disponible, usará la siguiente.

Fuentes comunes

  • Arial
  • Verdana
  • Times New Roman
  • Courier New
  • Georgia

Estas fuentes suelen estar disponibles en la mayoría de los sistemas.

¿Cómo usar fuentes de Google en CSS?

Google ofrece una biblioteca gratuita llamada Google Fonts que permite usar diferentes tipos de letra en una página web.

Paso 1: Entrar al sitio

Primero debes ingresar a la página:

https://fonts.google.com

Luego eliges la fuente que deseas utilizar.

Paso 2: Copiar el enlace

Google Fonts proporciona un enlace que debe colocarse dentro de la etiqueta head del archivo HTML. 

<link href="https://fonts.googleapis.com/css2?family=Roboto&display=swap" rel="stylesheet">

Paso 3: Usar la fuente en CSS

Después de agregar el enlace, puedes usar la fuente en tu archivo CSS con la propiedad font-family. 

body {
  font-family: 'Roboto', sans-serif;
}

Ahora todo el texto de la página utilizará la fuente Roboto.
